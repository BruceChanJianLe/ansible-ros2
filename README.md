# ANSIBLE ROS2

## Usage
Installing ROS2
```bash
ansible-pull -U https://github.com/brucechanjianle/ansible-ros2 -e "ros2_distribution=foxy" -K
```

For Docker
```bash
ansible-pull -U https://github.com/brucechanjianle/ansible-ros2 -e "ros2_distribution=foxy"
```
